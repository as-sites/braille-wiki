## Alphabet

Each braille cell is made up of up to six raised dots arranged in a 3 × 2 grid.
The dots are numbered 1–6 from top to bottom, left column first.

```
Dot layout:

  1 • • 4
  2 • • 5
  3 • • 6
```

### Example: the letter A

In UEB, the letter **a** is represented by dot 1 only:

```braille
⠁
```

## Contractions

UEB includes a set of contractions that shorten common words and letter groups.

| Contraction | Meaning      |
|-------------|--------------|
| `⠜`         | ar           |
| `⠉`         | c / ch       |
| `⠮`         | the          |
| `⠁⠝⠙`       | and          |
