# Braille Documentation

Welcome to the Braille Documentation site.

This site covers two major braille codes used in English-speaking countries:

- **[Unified English Braille (UEB)](ueb/introduction.md)** — the standard literary and technical braille code.
- **[Nemeth Braille](nemeth/overview.md)** — a braille code for mathematics and science notation.

For a quick lookup of characters, see the **[ASCII Braille Reference Table](reference/ascii-braille-table.md)**.
